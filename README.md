# Sale Dashboard Analysis

## Dashboard Overview

This dashboard provides an analysis of sales performance, orders,
quantities, payments, profits, products, and regional performance.

## Dashboard Sections

### Filters

-   Quarter selection: Qtr 1, Qtr 2, Qtr 3, Qtr 4
-   Store Location filter
-   Product filter

### Visualizations

-   Sales by Payment Method
-   Top Five Product by Profit
-   Profit by Region
-   Total Sales and Profit by Month
-   Product performance analysis

## Analysis Summary

The dashboard tracks sales trends, profit performance, payment methods,
product profitability, and regional results for business analysis.
